# Telegram Bot Demo (Educational Project)


## Overview
This repository contains a simple Telegram bot implemented using **Google Apps Script** and **Telegram Bot API**.


The bot accepts an 8-digit number, performs a basic arithmetic calculation, and returns the result.


This project is intended **solely for educational and demonstration purposes**.


---


## Features
- Telegram webhook integration
- Input validation
- Simple numeric calculation
- Error handling


---


## Disclaimer
⚠️ **Important Notice**


- This bot does **NOT** predict the future, health outcomes, or life events.
- All responses are purely mathematical and have **no real-world meaning**.
- The project does not collect, store, or analyze personal data.


---


## Security Notes
- **DO NOT commit real Telegram bot tokens or webhook URLs**.
- Use Google Apps Script properties or environment variables to store secrets.
- The values in the source code are placeholders only.


---


## Usage
1. Create a Telegram bot using [@BotFather]
2. Deploy a Google Apps Script as a Web App
3. Set your webhook URL
4. Send an 8-digit number to the bot
